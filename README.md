# 🐶 Dog Studio — Interactive 3D Web Experience

A modern, immersive **3D web experience** built with **Three.js and React.js**, designed to demonstrate advanced frontend development skills, real-time 3D rendering, and interactive UI/UX design.

---

## 🔗 Live Demo
👉 https://dogstudio-threejs-z42p.onrender.com

---

## 📌 Overview

Dog Studio is a visually rich, interactive 3D environment that showcases a dog-themed studio experience built using WebGL. The project focuses on performance-optimized rendering, smooth animations, and engaging user interaction.

This project was developed as part of my frontend and 3D web development learning journey.

---

## 🛠 Tech Stack

- React.js
- Three.js (WebGL)
- JavaScript (ES6+)
- HTML5 / CSS3
- Render (Deployment)
- WebGL Rendering

---

## ✨ Key Features

- Real-time 3D rendering using Three.js
- Smooth camera movement and animations
- Interactive 3D scene experience
- Responsive design across devices
- Optimized performance for WebGL rendering
- Clean and modular frontend architecture
